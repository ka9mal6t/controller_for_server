## Run Locally  

Clone the project  

~~~bash  
  git clone https://github.com/ka9mal6t/controller_for_server
~~~

Go to the project directory  

~~~bash  
  cd client
~~~

Install dependencies  

~~~bash  
pip install -r requirements.txt
~~~

Start the server  

~~~bash  
python main.py
~~~
